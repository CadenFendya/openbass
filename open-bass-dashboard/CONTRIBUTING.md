# Contributing

Open Bass Dashboard is designed to be open-source and contributor friendly.

## Good first issues

- Add regional bait color presets
- Improve water temperature estimation
- Add more waterbody profile fields
- Build editable map markers
- Add USGS service integration
- Improve accessibility and outdoor readability

## Development

```bash
npm install
npm run dev
```

## Coding principles

- Keep recommendations explainable
- Prefer deterministic scoring before black-box AI
- Keep the app mobile-first
- Avoid vendor lock-in
- Make offline behavior reliable
- Comment logic that affects fishing recommendations
