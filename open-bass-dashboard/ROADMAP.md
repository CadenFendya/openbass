# Roadmap

## Milestone 1: Core PWA
- Mobile-first UI
- Open-Meteo integration
- Bass Activity Score
- Bait recommendation engine
- Water estimation engine
- Saved spots, gear, and logbook

## Milestone 2: Maps
- Leaflet interactive map
- Editable pins for grass, pads, docks, laydowns, rocks, and access points
- Offline map tile caching where legally supported
- Satellite layer options

## Milestone 3: River intelligence
- USGS gauge support
- Flow, gauge height, and river temperature
- River-specific safety and fishability scoring

## Milestone 4: Better AI layer
- Natural language fishing assistant
- Explainable pattern summaries
- Personal catch history pattern mining
- Local model support for offline explanations

## Milestone 5: Native app path
- Capacitor wrapper
- iOS and Android builds
- Push reminders for prime windows
- App Store / Play Store readiness
